# Sistema de Inventario - MED CENTRO MÉDICO

Sistema de gestión de inventario desarrollado para la clínica MED CENTRO MÉDICO. Permite el control de inventario, gestión de tickets de solicitud y seguimiento de productos.

## Características Principales

### 1. Sistema de Autenticación
- Login con diferentes roles (Almacén, Enfermería, Administrador)
- Interfaz adaptada según el rol del usuario
- Sesiones persistentes con localStorage

### 2. Dashboard Principal
- Estadísticas en tiempo real
- Actividad reciente del sistema
- Alertas de stock bajo y tickets pendientes

### 3. Gestión de Inventario
- Registro de productos con código de barras
- Control de entradas y salidas
- Alertas de stock mínimo
- Ubicación en almacén

### 4. Sistema de Tickets
- Creación de tickets por enfermería
- Especificación de paciente y habitación
- Prioridades (Baja, Media, Alta, Urgente)
- Validación de stock disponible

### 5. Seguimiento de Tickets
- Visualización de todos los tickets
- Filtros por estado y búsqueda
- Actualización de estados (Pendiente, En proceso, Completado)
- Estadísticas de rendimiento
